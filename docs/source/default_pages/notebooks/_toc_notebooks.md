# Notebooks

Here you find examples of documentation which is written with Jupyter Notebooks. The source location of the notebooks is in `Equayes/equayes/docs/source/notebooks/Example_Notebook.ipynb` demonstrate how to **use** this equayes python package. The other ones show how to write nice documentation with myst syntax.  

```{note}
These static notebooks are real executed examples which are translated into sphinx-doc readable markdown with the package called [myst-nb](https://myst-nb.readthedocs.io/en/latest/).
Once installed correctly, the usage of this equayes should be the same in **.py** files and in **.ipynb** notebooks like these doc-examples.
```
